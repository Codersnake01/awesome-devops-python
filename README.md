## 🎯 Cómo Usar Este Repositorio

### Para Aprender
1. Comienza con **`python-for-devops`** - Es el mejor punto de entrada
2. Explora **`devops-exercises`** para preguntas y conceptos
3. Practica con **`homelab`** o **`kubernetes-workshop`**

### Para Contribuir
1. Elige un repositorio que te interese
2. Lee el archivo `CONTRIBUTING.md`
3. Busca issues con etiquetas: `good first issue`, `help wanted`, `beginner-friendly`
4. Fork el repositorio y crea un branch con tus cambios
5. Haz un Pull Request

### Para Guardar Estas Referencias
- ⭐ **Haz Star** a este repositorio
- 📌 **Marca** los repositorios que te interesen
- 📋 **Copia** esta lista para tu referencia personal

1. **Comienza pequeño**: Busca issues de baja dificultad
2. **Lee el código**: Entiende cómo funciona antes de modificar
3. **Sigue los estándares**: Respeta el estilo de código del proyecto
4. **Documenta**: Escribe buenos mensajes de commit y PR descriptions
5. **Sé respetuoso**: La comunidad open-source es colaborativa
